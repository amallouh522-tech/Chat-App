# Chati - Project Features & Future Plans

## Current Features

### Authentication & User Management
- ✅ User Login
- ✅ User Signup
- ✅ User Logout
- ✅ Session Management
- ✅ User Profile (RID-based identification)
- ✅ User Statistics (Posts_num, Followers, Following)

### Posts & Social Features
- ✅ Create Posts (with title, text, and image upload)
- ✅ View All Posts (ordered by ID DESC)
- ✅ Like/Unlike Posts
- ✅ Post Likes Counter
- ✅ View User Posts on Profile

### Chat & Messaging
- ✅ Create Chat Rooms
- ✅ Real-time Messaging (Socket.io)
- ✅ Chat List (fetch user's chats)
- ✅ Send Messages
- ✅ Load Chat Messages
- ✅ Chat Members Management

### Profile Features
- ✅ View Own Profile
- ✅ View Other Users' Profiles
- ✅ Profile Posts Display
- ✅ User Information Display

### File Upload
- ✅ Image Upload for Posts
- ✅ Static File Serving

## Technical Stack

### Frontend
- React (with React Router)
- Vite
- Socket.io Client

### Backend
- Node.js
- Express.js
- MySQL Database
- Socket.io Server
- Multer (File Upload)
- Express Session

### Database Tables
- `users` - User accounts and profiles
- `posts` - User posts
- `likes` - Post likes tracking
- `Chats` - Chat rooms
- `msgs` - Messages
